# ServerController

一个用于启动和管理其他进程的Java应用程序。

## 功能特性

- 模块化进程管理
- 基于配置文件的模块定义
- 自动启动和自动重启
- 实时日志输出和查看
- 交互式命令行界面
- 强制结束进程功能
- 模块输出切换视图


## 编译和运行

### 编译项目

```bash
./gradlew build
```

### 运行程序

```bash
./gradlew run
```

或者编译后直接运行：

```bash
java -jar build/libs/ServerController-1.0-SNAPSHOT.jar
```


## 开发说明

本项目采用模块化设计，主要包含以下核心组件：

1. **模块管理服务** - 负责模块的注册、启动、停止等操作
2. **进程管理器** - 管理底层进程的生命周期
3. **日志服务** - 收集和存储模块的输出日志
4. **自动启动服务** - 处理模块的自动启动逻辑
5. **命令系统** - 提供交互式命令行界面

所有组件都通过依赖注入的方式进行组合，便于扩展和维护。
